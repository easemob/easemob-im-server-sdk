All notable changes to this project will be documented in this file.

## [0.3.0] - 2021-07-12

### Added

- 可自定义 baseURI
- 可设置 HTTP client proxy
- 可对用户属性 metadata 做增删改查

### Changed

- Upgraded Reactor to 2020.0.4
- Upgraded Netty to 4.1.59.Final
